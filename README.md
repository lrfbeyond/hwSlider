#[hwSlider](http://www.helloweba.com/view-blog-372.html)

hwSlider是一个基于jQuery的响应式的内容滑动插件。

<p align="center">
  <img src="./jquery.hwslider.jpg" />
</p>

DEMO: [http://www.helloweba.com/demo/2016/hwSlider/plugin.html](http://www.helloweba.com/demo/2016/hwSlider/plugin.html "hwSlider")

##hwSlider特性

<ol>
	<li>多个参数定制，满足不同项目需求。</li>
	<li>支持移动端触控滑动。</li>
	<li>支持图文混排，支持各种html元素。</li>
	<li>响应式自适应屏幕尺寸。</li>
	<li>同一页面支持多个滑动切换。</li>
	<li>轻量级的，压缩版不到4KB。</li>
	<li>支持所有现代浏览器，支持ie8+。</li>
</ol>
